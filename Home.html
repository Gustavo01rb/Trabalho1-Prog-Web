Jogos para todos os tipos de jogadores

O sucesso que os jogos eletrônicos, ou games, vêm fazendo em todas as faixas etárias é inegável e grande parte deste sucesso só pode ser entendido com base em uma análise dos seus componentes e interrelacionamentos. Por meio do estudo sobre o perfil dos jogadores, assim como a análise das inter-relações entre os diversos fatores que fazem os games cativantes, pode-se realmente entender como as técnicas de engajamento utilizadas pelos desenvolvedores de games para redes sociais atuam. Não se deve entender, contudo, que o presente artigo se restringe à indústria de games. O maior objetivo aqui é demonstrar como os jogadores são motivados a jogar e ponderar como empregar este tipo de técnica em outras áreas.
